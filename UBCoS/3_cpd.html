2 Data process

2.1 Uppsala 1st generation cohort

2.1.1 Wave 1

2.1.1.1 Study-specific variable description

Two variables are taken from the UBCoS dataset:

  • Name: cpd_firstdate
  • Table name: ubcosG1W1
  • Description: Date of first diagnosis of cpd

  • Name: cpd_firstage
  • Table name: ubcosG1W1
  • Description: Age of first diagnosis of cpd

2.1.1.2 Harmonisation algorithm

For the variable h_cpd, value 0 is assigned when date of first diagnosis of cpd is later than 1st November 1960, the approximate date of the census registry, or never, and value 1 when prior to that date.

R script:

# 'sds': study dataset
# 'hds': harmonised dataset
hds <- tibble(id=sds[[1]], h_cpd=0)

for (i in 1:length(sds[[2]])) {
  if (!is.na(ubcosG1W1$cpd_firstdate[i]) & ubcosG1W1$cpd_firstdate[i] <= "1960-11-01")
         hds$h_cpd[i] <- 1
}

For the variable h_cpd_age, the same age is assigned when the first diagnosis of cpd is before than 1st November 1960, otherwise the value 994 (not applicable) is assigned.

R script:

# 'sds': study dataset
# 'hds': harmonised dataset
hds$h_cpd_age <- sds[[3]]

for (i in 1:length(sds[[2]])) {
  if (hds$h_cpd[i] != 1)
    hds$h_cpd_age[i] <- 994   # not applicable
}
